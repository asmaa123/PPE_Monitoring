# Data Science Notebooks Portfolio

A collection of machine learning and data analysis notebooks covering classification, clustering, association rules, and regression use cases.

## Project Contents

- `asmaa_mz ChurnModel.ipynb`: customer churn modeling workflow.
- `Asmaa_mz_clustering.ipynb`: clustering experiments and evaluation.
- `apriori alogrithm.ipynb`: association rule mining with Apriori.
- `home pricing asmaa_mz.ipynb`: home price prediction model.
- `insurance_Asmaa_mz.ipynb`: insurance-related predictive analysis.
- `Tanzania Predict Pump Asmaa_mz.ipynb`: notebook for pump status prediction.
- `Tanzania Predict Pump Asmaa_mz.html`: exported notebook report.

## Requirements

- Python 3.9+
- Jupyter Notebook or JupyterLab

Install common dependencies:

```bash
pip install jupyter pandas numpy matplotlib seaborn scikit-learn
```

## Usage

1. Clone the repository.
2. Create and activate a Python virtual environment.
3. Install dependencies.
4. Start Jupyter:

```bash
jupyter notebook
```

5. Open and run any notebook in this repository.

## Notes

- Notebooks may use dataset paths local to the original environment; update paths before running.
- For reproducibility, consider adding `requirements.txt` in a future update.
